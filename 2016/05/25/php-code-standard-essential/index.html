文字说明

基础：

  1. 只允许<?php或<?=，不允许<?短标记；
  2. 如果是纯PHP代码，则结尾不加?>标记（官方推荐，防止意外输出）；
  3. 行尾不要有空格；
  4. 文件使用UTF-8编码，去除BOM；
  5. 使用4个空格代替Tab缩进（在IDE里设置缩进选项）；
  6. 所有类、方法或变量等的命名都以英文意义命名，名称要直观有意义；

命名空间：

  1. 命名空间要跟目录结构保持一致；
  2. 命名空间声明在第一行；

类：

  1. 文件名跟类名一致；
  2. 每个单词首字母大写；
  3. 大括号新起一行；
  4. 类注释在class上面，非必需；

类常量和属性：

  1. 类常量全大写，用下划线分隔；
  2. 类属性全小写，下划线分隔；
  3. 如果是protected或private则以下划线开头；

方法：

  1. 第一个单词小写，之后每个单词首字母大写；
  2. protected或private方法名以下划线开头；
  3. 大括号新起一行；
  4. 方法跟方法之间至少有一行的分隔空间；
  5. 方法说明注释；

函数和匿名函数：

  1. 函数命名全小写，下划线分隔；
  2. 函数的大括号新起一行；
  3. 匿名函数大括号在本行；

变量和常量：

  1. 变量命名全小写，下划线分隔；
  2. 常量命名全大写，下划线分隔；
  3. 字符串使用单引号，除非包含变量或其它特殊情况；
  4. 字符串连接的点，两边加空格；

数组：

  1. 短数据写在一行；
  2. 长数据写多行，一行一个元素，箭头两边加上空格；

控制：

  1. 大括号在同一行开始，结束大括号在新一行；
  2. 都要加上大括号，哪怕只有一行语句；
  3. 条件声明，括号两边要加空格，比如 if ($a == 1) {
  4. 等号赋值或双等号判断等两边要有空格；
